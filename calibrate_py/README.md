# calibrate_py

Python port of the C# `Calibrate` application. Uses the provided `Python_LJM_2025_04_24` folder for LabJack LJM bindings.

Quick start:

- Ensure `Python_LJM_2025_04_24` is available in your PYTHONPATH or run from workspace root so the local package is importable.
- Install required packages: `pip install -r calibrate_py/requirements.txt`
- Run: `python -m calibrate_py.calibrate` from the workspace root.
